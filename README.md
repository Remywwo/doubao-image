## doubao-image

**Author:** remywwo
**Version:** 0.0.1
**Type:** tool

### Description

This Dify tool plugin generates images or videos from text prompts with Doubao generation models.

After installation, configure the Doubao `api_key` and optionally a default API model ID.

- `doubao-image`: uses Seedream models for image generation and Seedance models for video generation.
